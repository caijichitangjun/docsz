# vscode中好用的插件

## Chinese (Simplified) 
    使你的浏览器变成中文的界面

## Vetur
    编写vue十分友好，包括代码提示、问题修复等功能，强烈建议安装

## Markdown Preview Enhanced
    MD文件实时更新插件，即一边书写，一边可查看到结果
